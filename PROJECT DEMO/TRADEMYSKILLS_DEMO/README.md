# TradeMySkills Demo

This is a lab-style ENSE 281 demo built with:
- Node.js
- Express
- EJS
- Bootstrap
- Basic MVC folder structure

## Demo Pages
- `/` Home / landing page
- `/login` Login page
- `/register` Sign up page
- `/dashboard` Explore skills dashboard
- `/skills/1` Skill details page
- `/skills/new` Post skill form
- `/messages` Inbox / chat demo

## How to Run
1. Open a terminal in this folder.
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Notes
- This is a front-end / routing demo only.
- No database is required.
- The forms redirect to demo pages so the team can show a clean in-class prototype.
